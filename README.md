# GDG Noida Agentic Premier League (APL) 2026

**Big Plays. Big Builds.** - A 4-hour vibe coding challenge at Moire Cafe, Noida on 31st May 2026.

## About the Event

GDG Noida APL 2026 is a premier coding challenge where developers build innovative solutions in just 4 hours. Participants showcase their skills, creativity, and problem-solving abilities across multiple challenge tracks.

## Features

### For Participants
- **Quick Registration** - Sign up with your GitHub repository in under 2 minutes
- **Live Leaderboard** - Real-time rankings updated automatically
- **Multiple Challenge Tracks** - Choose from 4 exciting problem statements
- **Fair Judging System** - Automated evaluation based on commit patterns and code quality

### For Audience
- **Live Stream Projector** - Watch the action unfold in real-time
- **Interactive Reactions** - Send flying emojis during presentations using your phone
- **Live Scoreboard** - Track participant rankings as they're judged

### For Organizers
- **Admin Dashboard** - Control registration and judging with one click
- **Google Sheets Integration** - All data stored in familiar spreadsheet format
- **Automated Judging** - System evaluates GitHub repositories every 2 minutes
- **Real-time Updates** - Leaderboard refreshes automatically

## How It Works

1. **Participants register** with their GitHub repository URL
2. **Coding begins** - 4 hours of focused building
3. **Auto-judging runs** - System analyzes commit patterns continuously
4. **Leaderboard updates** - Scores appear automatically every 2 minutes
5. **Audience engages** - Live reactions during final presentations

## Judging Criteria

- **Commit Frequency (40%)** - Consistent coding activity throughout the challenge
- **Timing Distribution (30%)** - Well-paced development, not last-minute pushes
- **Message Quality (20%)** - Clear, descriptive commit messages
- **Authenticity (10%)** - Genuine building behavior detection
